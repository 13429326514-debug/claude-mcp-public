# Claude Code 启动脚本

这个目录包含用于快速启动不同项目 Claude Code 的批处理脚本。

## 📋 可用脚本

| 脚本 | 用途 | 默认项目路径 |
|------|------|-------------|
| **work-assistant.bat** | 工作助手 | `G:\claude\plan_helper` |
| **dev-assistant.bat** | 开发助手 | `G:\claude\dev_helper` |
| **data-assistant.bat** | 数据助手 | `G:\claude\data_helper` |
| **game-assistant.bat** | 游戏助手 | `G:\claude\game_dev_helper` |

## 🚀 使用方法

### 方法 1：直接双击运行

1. 找到对应的 `.bat` 文件
2. 双击运行
3. 会自动启动 Claude Code 并切换到对应项目目录

### 方法 2：创建桌面快捷方式

1. 右键点击 `.bat` 文件
2. 选择 "发送到" → "桌面快捷方式"
3. 可选：重命名快捷方式

### 方法 3：添加到启动项（推荐）

1. 按 `Win + R` 打开运行对话框
2. 输入 `shell:startup` 并回车
3. 将常用的 `.bat` 文件复制到此文件夹
4. 每次开机后会自动显示

## ⚙️ 自定义配置

每个脚本都会在指定目录启动 Claude Code。如需修改项目路径：

1. 右键点击 `.bat` 文件，选择 "编辑"
2. 找到这行：
   ```batch
   cd /d "G:\claude\your_project"
   ```
3. 修改为你的项目路径
4. 保存文件

## 📝 脚本功能

每个启动脚本会：

1. ✅ 设置 UTF-8 编码
2. ✅ 设置终端窗口大小（140x100）
3. ✅ 切换到项目目录
4. ✅ 启动 Claude Code（跳过权限检查）
5. ✅ 完成后暂停，等待按键关闭

## 🔧 高级配置

### 修改窗口大小

编辑脚本中的这一行：
```batch
mode con cols=140 lines=100
```

- `cols`：窗口宽度（字符数）
- `lines`：窗口高度（字符数）

### 移除权限检查

脚本使用 `--dangerously-skip-permissions` 参数跳过权限检查，适用于：
- 可信的工作环境
- 个人开发机器
- 熟悉 Claude Code 操作的用户

如需启用权限检查，删除此参数即可。

## 💡 使用技巧

### 快速切换项目

为不同项目创建多个快捷方式，放在桌面或任务栏，快速切换。

### 命名快捷方式

重命名快捷方式以更清晰地标识项目：
- `工作助手.lnk`
- `开发助手.lnk`
- `数据助手.lnk`
- `游戏助手.lnk`

### 自定义图标

1. 右键点击快捷方式
2. 选择 "属性"
3. 点击 "更改图标"
4. 选择自定义图标

## ⚠️ 注意事项

- 确保项目路径存在
- 确保已安装 Claude Code
- 建议将脚本放在固定位置，避免移动后路径失效

## 🆘 故障排除

### 问题：双击后闪退

**原因：** 项目路径不存在

**解决：** 编辑脚本，修改 `cd /d` 后的路径为正确路径

### 问题：无法启动 Claude Code

**原因：** Claude Code 未安装或未加入 PATH

**解决：**
1. 重新安装 Claude Code
2. 或使用完整路径替换 `claude` 命令

### 问题：编码显示异常

**原因：** 终端编码不支持中文

**解决：** 脚本已设置 UTF-8 编码，如仍有问题，检查终端设置
